Button

Button elements, links with a button class and input elements should all render exactly the same.

Group buttons into a button-list to ensure correct spacing between buttons.

Examples

Default Buttons

Link 
<div class="button-list">
    <button>Button</button>
    <a class="button" href="#">Link</a>
    <input type="button" value="Input">
    <input type="submit" value="Submit">
</div>

Different Styles

<div class="button-list">
    <button><i class="material-icons">menu</i><span>Icon</span></button>
    <button><i class="material-icons">close</i></button>
    <button class="button-primary">Primary</button>
    <button class="button-active">Active</button>
    <button disabled>Disabled</button>
    <button class="button-danger"><i class="material-icons">delete_forever</i><span>Danger</span></button>
    <button>Japanese ボタン</button>
</div>

Japanese Specific Font

<div lang="ja-JP">
    <button>Japanese ボタン</button>
</div>
