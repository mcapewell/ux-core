Button Group

Group a series of buttons together on a single line with no spacing between each one.

Examples

Pagination

An example of a button-group is a simple pagination component.

<div class="button-group">
    <button disabled><i class="material-icons">chevron_left</i><span>Previous</span></button>
    <button class="button-active">1</button>
    <button>2</button>
    <button>3</button>
    <button>...</button>
    <button>99</button>
    <button><span>Next</span><i class="material-icons">chevron_right</i></button>
</div>

Toolbar

Combine sets of button-group into a button-toolbar.

<div class="button-toolbar">
    <div class="button-group">
        ...
    </div>
    <div class="button-group">
        ...
    </div>
</div>
