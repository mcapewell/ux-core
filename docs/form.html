Form

Default forms will have full-width, block level labels, inputs, selects, and textareas.

Examples

Text Field

<div>
    <label>Default TextField</label>
    <input type="text" value="Some value" />
</div>
<div>
    <label>TextField with Placeholder</label>
    <input type="text" placeholder="Placeholder" value="" />
</div>
<div>
    <label>Disabled TextField</label>
    <input type="text" value="Some value" disabled />
</div>
<div>
    <label>Multiline TextField</label>
    <textarea rows="3">Some text</textarea>
</div>
<div>
    <label>Disabled Multiline TextField</label>
    <textarea rows="3" disabled>Some text</textarea>
</div>

Select

<div>
    <label>Default Select</label>
    <select>
        <option>Please select</option>
        <option>Option 1</option>
        <option>Option 2</option>
    </select>
</div>
<div>
    <label>Disabled Select</label>
    <select disabled>
        <option>Please select</option>
        <option>Option 1</option>
        <option>Option 2</option>
    </select>
</div>
